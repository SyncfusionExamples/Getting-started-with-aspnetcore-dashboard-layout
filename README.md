# Getting-started-with-aspnetcore-dashboard-layout

**Repository Description**  
This repository contains an ASP.NET Core sample that demonstrates how to build static and dynamic dashboards using the Syncfusion ASP.NET Core Dashboard Layout component.

The ASP.NET Core Dashboard Layout is a grid‑structured layout component that helps create dashboard interfaces composed of interactive panels. Dashboard panels are the fundamental building blocks of a dashboard and can be added either programmatically or dynamically at runtime. These panels can host UI components and support resizing, reordering, dragging, dropping, removal, and insertion, allowing users to place components at desired positions within a grid‑based layout.

## Project Overview
The purpose of this project is to help developers understand how to create flexible and configurable dashboards in ASP.NET Core using the Syncfusion Dashboard Layout component. It demonstrates how panel‑based layouts can be managed dynamically to build responsive data visualization or monitoring interfaces.

## Features
- Integration of the Syncfusion ASP.NET Core Dashboard Layout component  
- Grid‑based dashboard layout system  
- Dashboard panels that support resizing and reordering  
- Drag‑and‑drop functionality for dashboard panels  
- Dynamic addition and removal of panels  
- Support for both static and dynamic dashboard configurations  

## Prerequisites
Ensure the following tools are installed before running this project:
- Visual Studio  
- .NET SDK compatible with ASP.NET Core  

## Installation and Running the Application
1. Clone the repository.
```bash
git clone https://github.com/SyncfusionExamples/Getting-started-with-aspnetcore-dashboard-layout.git
```
2. Open the solution file in Visual Studio.
3. Restore NuGet packages by rebuilding the solution.
4. Build and run the application.
5. Open the application URL shown in the browser to view the Dashboard Layout output.

## Usage
Run the application and interact with the dashboard panels to:
- Drag panels and rearrange their positions  
- Resize panels dynamically  
- Add or remove panels at runtime  
- Organize UI components within the dashboard grid
